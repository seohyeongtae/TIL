### 데이터 분석 과정

> 데이터 분석 설계 -> 데이터 준비 -> 데이터 가공 -> 데이터 분석 -> 결론 도출

- 데이터 분석 설계 : 방향성 기획 / 방법론 검토 / 가설 설정
- 데이터 준비 : 데이터 불러오기 / 형태 파악하기
- 데이터 가공 : 추출 및 정제 / 파생 변수 생성 / 데이터 병합
- 데이터 분석 : 통계 분석 / 그래프 및 시각화
- 결론 도출 : 분석 결과 해석 / 분석 결과 정리
